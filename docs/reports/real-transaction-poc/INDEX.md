# Real Anonymized Transaction PoC Package

> Public-safe sample generated from one anonymized F-ACC transaction card.

## Summary

- Domain: KIFRS1116
- Target service-line: F-ACC Accounting Advisory / F-S support
- Review pack status: automated
- Queue records: 1 eval seed candidate

## Files

1. `anonymized-input-card.md` - sanitized structured transaction card
2. `review-pack.md` - F-ACC review pack generated by the existing 1116 pipeline
3. `feedback-queue.jsonl` - public-safe feedback queue record
4. `feedback-queue-report.md` - queue report for reviewer correction
5. `package-manifest.json` - machine-readable package manifest

## Boundary

- This package uses synthetic anonymized facts for public verification.
- It does not include copied contract text, customer identifiers, private filings, parsed standards, embeddings, or workpaper payloads.
- The review pack is a decision-prep draft, not a final accounting conclusion.
