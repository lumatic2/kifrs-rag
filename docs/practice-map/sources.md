## 대분류 골격 (확정)

| 코드 | 대분류 | 비고 |
|---|---|---|
| A | 감사·인증 (외부감사, 내부회계관리제도) | Big4·중견 법인 핵심, 회계사 대부분의 주 업무 |
| B | 결산·재무제표 작성 지원 | 기업 측 업무이지만 회계사가 지원·자문 (기장 포함 — 로컬 비중 큼) |
| C | 세무 (신고·조정·자문) | **자동화 실험은 sibling `tax-agent` 담당** — 지도 표기만 |
| D | 재무자문 (Deal/FAS: 실사·가치평가·M&A) | 건별 프로젝트 |
| E | 기타 인증·컨설팅 (내부회계 구축, 사업비 검증, ESG) | 성장 영역 |

근거: 공인회계사 업무의 전통적 3분류(감사/세무/경영자문)에 실무 관찰(기장·결산 지원의 로컬 비중,
Deal 부문 독립성, 내부회계·ESG 인증 성장)을 반영해 5분류로 확장. Big4는 A·D·E 중심, 로컬 법인은
A(중소 감사)·B·C 중심 — 두 유형을 같은 지도에 담기 위해 B를 독립 대분류로 뒀다.
