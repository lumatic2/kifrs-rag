# Horizon: Field Feedback Ready Demo

> Status: closed
> Created: 2026-07-05
> Sequence: `docs/horizons/accounting-intelligence-expansion.md`
> Previous: `docs/horizons/multi-authority-runtime-integration.md`

## Goal

최신 runtime evidence demo를 회계사 1명에게 보여주고 피드백을 받을 수 있는 package로 정리한다. 목표는
새 기능 구현이 아니라, demo brief, feedback questionnaire, known limitations, smoke checklist를 현업
피드백 가능한 형태로 맞추는 것이다.

## Why next

`multi-authority-runtime-integration`은 ingestion/evidence manifest를 review pack, statement draft, answer
boundary, demo bundle까지 연결했다. 이제 이 결과물을 회계사가 읽고 판단할 수 있게 설명해야 한다.
Objective의 중간 관문인 "현업 회계사 1명 이상이 실제 업무 사례를 이 도구로 처리해보고 피드백"으로
가기 위한 바로 전 단계다.

## Candidate Milestones

### FF1. Demo Brief Refresh

최신 runtime evidence demo 기준으로 10분 설명 brief를 갱신한다.

Deliverable:

- runtime-aware demo brief

### FF2. Feedback Questionnaire Refresh

질문지를 정확성, 유용성, 검토 부담, 위험, 다음 PoC 후보로 재구성한다.

Deliverable:

- feedback questionnaire

### FF3. Known Limitations and Human-Review Boundary

무엇을 자동화했고 무엇을 자동화하지 않았는지 현업 피드백 전용 경계 문서로 정리한다.

Deliverable:

- limitations/human-review boundary

### FF4. Feedback Package Smoke and Close

demo command, 주요 산출물, 피드백 패키지 경로를 검증하고 horizon을 닫는다.

Deliverable:

- feedback package index
- close report
